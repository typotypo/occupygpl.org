Prefer projects using a permissive license!

Don't use a library or component that is GPL licensed or a lot of people won't be able to use your project!

(Re-)License your projects using a permissive license!

License your projects using a permissive license like the MIT, BSD or zlib license. If you have existing GPL projects think about relicensing them. Please be aware that the other contributors also need to agree to the relicensing.

Let projects know you don't want to use the GPL anymore!

You want to use a project but it's GPL licensed? Try to open an issue or contact a maintainer about a possible license change.

Help new open source developer understand why the GPL is bad!

A lot of young open source developers are using the GPL as their open source license without even knowing the consequences. Help them understand in which way the GPL may restrict others and tell them about permissive licenses such as the MIT, BSD or zlib license.
